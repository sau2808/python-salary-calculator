# Python Salary Calculator

A simple Python project for freshers to practise:

- Python functions
- User input
- Dictionary
- Percentage calculations
- Exception handling
- Virtual environments
- Git and GitHub

## Salary Rules

- HRA: 20% of basic salary
- DA: 10% of basic salary
- PF: 12% of basic salary

## Run the Project

Create a virtual environment:

```bash
python -m venv venv
```
